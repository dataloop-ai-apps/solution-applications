# *Metadata Enrichment* Pipeline Example: Image Preprocessing

### Introduction:

This solution provides a pipeline for automatically enriching image metadata with EXIF data and computing Blurriness 
and Darkness scores. With this approach, you can efficiently assess image quality at scale, enabling advanced filtering 
and quality-based workflows. This example includes a prepared Wildlife Images Dataset from NVIDIA.

### Installation:

To install and use the example, follow these steps:

* Open the pipelines page and select Create Pipeline.
* Select Use a Template from the dropdown list.
* In the search bar, type `Metadata Enrichment`, select the template and click install.
* Once the template is installed, click on *Create Pipeline*.
