# *PII Anonymizer* Pipeline Example: PDF Preprocessing

### Introduction:

This solution provides a pipeline for anonymizing Personally Identifiable Information (PII) from PDF files. It 
automatically detects and redacts sensitive data such as names, addresses, and medical records. This example includes 
a prepared Medical Records Dataset to demonstrate the anonymization workflow.

### Installation:

To install and use the example, follow these steps:

* Open the pipelines page and select Create Pipeline.
* Select Use a Template from the dropdown list.
* In the search bar, type `PII Anonymizer`, select the template and click install.
* Once the template is installed, click on *Create Pipeline*.

### Usage:

For the complete documentation of the PII Anonymizer pipeline, please refer to
