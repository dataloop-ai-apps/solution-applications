# *Data Indexing* Pipeline Example: Clustering Visualization

### Introduction:

This solution provides a pre-indexed dataset with 2 feature sets, ready to explore using the Clustering tab. It uses 
the Retail Products Dataset with pre-extracted embeddings, allowing you to visualize data distribution, spot outliers, 
find duplicates, and semantically search your data — without any additional setup.

### Installation:

To install and use the example, follow these steps:

* Open the pipelines page and select Create Pipeline.
* Select Use a Template from the dropdown list.
* In the search bar, type `Data Indexing for Clustering`, select the template and click install.
* Once the template is installed, click on *Create Pipeline*.

### Usage:

After installation, navigate to the dataset and open the **Clustering** tab to explore the data. The 2 feature sets 
enable different visual perspectives on the dataset, allowing you to compare embeddings and discover patterns across 
the Retail Products data.

For the complete documentation on data indexing and clustering, please refer to
the [Dataloop Documentation](https://docs.dataloop.ai/)
